# Mercavejo - Design Brainstorm

## Contexto
Agência de marketing em Catanduva com foco em: estúdio de podcast, landing pages, tráfego pago, edição de vídeo, criação de artes, consultoria e gerenciamento de redes sociais. Cores predominantes: azul escuro e amarelo claro. Estética contemporânea.

---

## Resposta 1: Modernismo Corporativo com Dinâmica Geométrica
**Probabilidade: 0.08**

**Design Movement:** Modernismo corporativo contemporâneo com influências de design de produto (Apple, Stripe)

**Core Principles:**
- Minimalismo funcional com hierarquia clara
- Movimento sutil através de formas geométricas
- Contraste forte entre elementos (azul escuro vs. amarelo claro)
- Espaçamento generoso e respiração visual

**Color Philosophy:**
- Azul escuro (#0F3A7D ou similar) como cor primária - transmite confiança, profissionalismo e tecnologia
- Amarelo claro (#d4b67b ou #d4b67b) como accent - energia, criatividade, destaque de CTAs
- Branco/off-white como fundo principal - clareza e modernidade
- Cinza neutro para textos secundários

**Layout Paradigm:**
- Grid assimétrico com seções que alternam entre full-width e conteúdo centrado
- Uso de diagonais e ângulos sutis (clip-path) para quebrar a monotonia
- Seções com background colorido intercaladas com branco
- Sidebar ou cards flutuantes para destacar serviços

**Signature Elements:**
- Ícones geométricos minimalistas (linhas, formas abstratas)
- Gradientes sutis do azul para tons mais claros
- Dividers diagonais ou ondulados entre seções
- Botões com efeito hover que muda para amarelo

**Interaction Philosophy:**
- Transições suaves ao hover (0.3s ease-in-out)
- Botões que ganham profundidade com sombra ao hover
- Cards que se elevam ao passar o mouse
- Animações de entrada suave para elementos ao scroll

**Animation:**
- Fade-in suave para elementos ao entrar na viewport
- Slide-up para cards de serviços
- Rotate suave em ícones ao hover
- Pulse sutil em CTAs para atrair atenção

**Typography System:**
- Display: Fonte moderna e bold (ex: Poppins Bold 700) para títulos principais
- Heading: Poppins SemiBold 600 para subtítulos
- Body: Inter Regular 400 para textos longos
- Accent: Poppins Bold 700 para destaques em amarelo

## Resposta 2: Estética Criativa Dinâmica com Movimento Orgânico
**Probabilidade: 0.07**

**Design Movement:** Brutalismo digital com elementos orgânicos (Dribbble, design studios contemporâneos)

**Core Principles:**
- Tipografia grande e ousada como protagonista
- Formas orgânicas e curvas (não apenas ângulos retos)- Movimento constante e energia visual
- Sobreposição de elementos para criar profundidade

**Color Philosophy:**
- Azul escuro como base sólida e confiável
- Amarelo claro em grandes áreas para criar impacto visual
- Cores complementares sutis (laranja, roxo claro) para destaque
- Preto para contraste extremo em tipografia

**Layout Paradigm:**
- Overlapping sections com clip-path orgânico
- Tipografia que "invade" imagens
- Uso de espaço negativo criativo
- Seções com altura variável para ritmo visual

**Signature Elements:**
- Tipografia oversized e ousada
- Formas blob/orgânicas como separadores
- Ilustrações abstratas coloridas
- Elementos que se estendem além dos containers

**Interaction Philosophy:**
- Interações mais dramáticas e visíveis
- Elementos que se movem ao scroll (parallax)
- Hover effects que transformam elementos
- Animações que revelam conteúdo

**Animation:**
- Parallax ao scroll
- Scale e rotate ao hover
- Stagger animations para listas
- Morphing shapes entre seções

**Typography System:**
- Display: Fonte sans-serif ousada (ex: Clash Display) para impacto
- Heading: Poppins Bold para hierarquia
- Body: Inter para legibilidade
- Accent: Clash Display em amarelo para destaque

## Resposta 3: Elegância Minimalista com Foco em Conteúdo
**Probabilidade: 0.06**

**Design Movement:** Minimalismo escandinavo com toques de design editorial

**Core Principles:**
- Conteúdo é rei - design não compete com mensagem
- Tipografia limpa e hierarquizada
- Uso estratégico de cor (menos é mais)
- Proporções e alinhamento precisos

**Color Philosophy:**
- Azul escuro em elementos estruturais (header, footer, accents)
- Amarelo claro apenas em CTAs e destaques críticos
- Muito branco e cinza claro para respiro
- Monocromático com pops de cor

**Layout Paradigm:**
- Grid simétrico e previsível
- Margens e padding generosos
- Tipografia como elemento visual principal
- Seções bem definidas e separadas

**Signature Elements:**
- Linhas horizontais sutis como separadores
- Tipografia em escala variada como design
- Espaço branco abundante
- Imagens e ícones minimalistas

**Interaction Philosophy:**
- Interações sutis e não distrativas
- Hover effects discretos
- Transições suaves e previsíveis
- Foco em usabilidade

**Animation:**
- Fade-in suave
- Slide suave ao scroll
- Hover com mudança de cor sutil
- Transições de 0.2-0.3s

**Typography System:**
- Display: Playfair Display ou similar para elegância
- Heading: Inter SemiBold para clareza
- Body: Inter Regular para leitura
- Accent: Playfair Display em amarelo para destaque

---

## Design Escolhido: Modernismo Corporativo com Dinâmica Geométrica

Escolhi a **Resposta 1** por ser a mais adequada para uma agência de marketing profissional que precisa transmitir credibilidade enquanto mantém energia e criatividade. O contraste entre azul (confiança) e amarelo (criatividade) funciona perfeitamente para o posicionamento da Mercavejo.

### Implementação:
- **Cores:** Azul escuro (#0F3A7D), Amarelo claro (#d4b67b), Branco, Cinza neutro
- **Tipografia:** Poppins para títulos, Inter para corpo
- **Layout:** Grid assimétrico com seções dinâmicas
- **Elementos:** Ícones geométricos, dividers diagonais, gradientes sutis
- **Interações:** Transições suaves, hover effects elegantes
